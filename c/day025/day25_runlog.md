## 已掌握

| 函数 | 返回类型 | 作用 |
|------|---------|------|
| `show_menu()` / `show_scores()` | `void` | 打印菜单选项，不参与计算 |
| `print_scores(arr, size)` | `void` | 遍历打印数组每个元素 |
| `sum_scores(arr, size)` | `int` | 累加所有元素返回总和 |
| `average_scores(arr, size)` | `double` | 返回平均值，`(double)sum / size` 强转避免整数除法 |
| `max_scores(arr, size)` | `int` | 找最大值（从 `arr[0]` 开始比较） |
| `min_scores(arr, size)` | `int` | 找最小值（条件用 `<`，不是 `>`） |
| `count_pass(arr, size)` | `int` | 统计 >= 60 的及格人数 |
| `count_scores(arr, size)` | `int` | 统计 < 60 的不及格课程数 |

## 关键理解

- **菜单循环用 `do-while`**：因为菜单至少要先显示一次再读用户选择 ✅
- **`scanf` 不要加 `\n`**：`scanf("%d\n", ...)` 会卡住等额外输入，改成 `"%d"` 就好 ✅
- **void 函数不能 `return` 值**：`show_menu` 只打印，不返回数据 ✅
- **int 函数必须 `return`**：`count_pass` 算出 `counts` 后要用 `return counts` 送回 ✅
- **函数调用要接住返回值**：`int sum = sum_scores(arr, size);` 不是只调不用 ✅
- **平均分用 `(double)sum / size`**：整数除法会丢小数，强转一方为 `double` 再除 ✅
- **`int max = 0` 的陷阱**：如果成绩全负数，最大值会是 0（错误）；必须初始化为 `arr[0]` ✅
- **`min_scores` 条件用 `<` 不是 `>`**：找最小值的条件是 `arr[i] < min`，写成 `>` 会找到最大值 ✅
- **if 语句的 body 不能空着**：`if(...) { }` 判断了但没做事，变量永远不会变 ✅
- **无效选项分支不可少**：没有 `else`，输入 9 程序会直接退出而不是提示重试 ✅
