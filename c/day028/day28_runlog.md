## 已掌握

| 概念 | 说明 |
|------|------|
| `&` 取地址 | `&a` 得到变量 a 的内存地址，格式符 `%p` |
| `*` 声明指针 | `int *p = &a` 声明一个指针 p，存放 a 的地址 |
| `*` 解引用 | `*p` 访问 p 指向的那块内存的值 |
| 通过指针改值 | `*p = 20` 等价于 `a = 20` |
| 一个指针一次指向一个变量 | `int *p = &a` 后 `p` 只能指向 a，不能同时指向两个变量 |
| 指针交换需要两个指针 | `px→x`、`py→y`，通过 `temp` 倒换 `*px` 和 `*py` |

## 关键理解

- `int *p` 声明了指针，`p` 存地址，`*p` 取值 ✅
- `*p = 20` 和 `a = 20` 效果一样，因为 `p` 指向 `a` ✅
- `int *px = &x` 中 `*` 是贴在变量上的——每个指针变量都要单独写 `*` ✅
- 指针的值（地址）在赋值后不变，改变的是指向的内存内容 ✅
- `printf` 打印地址用 `%p`，打印值用 `%d` ✅
