## 学习要点

- 数组创建：`int arr[5];` 或 `int arr[5] = {1,2,3,4,5};`
- 下标从 0 开始，长度 5 的下标范围是 `0~4`
- 赋值语句必须写在函数内（不能在文件作用域赋值）
- 遍历用 `for (i = 0; i < 长度; i++)`
- 越界访问（`i <= 5`、`i >= -1`）读到垃圾值
