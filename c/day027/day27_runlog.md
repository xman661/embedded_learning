## 已掌握

| 概念 | 说明 |
|------|------|
| 二维数组声明 | `int arr[行][列]` |
| 嵌套循环遍历 | 外层 `i` 行号，内层 `j` 列号，`arr[i][j]` 访问 |
| 内存布局 | 按行连续存储（row-major order） |
| 矩阵格式打印 | 内层循环打每个元素，外层循环末尾 `printf("\n")` 换行 |
| 主对角线 | `arr[0][0] + arr[1][1] + arr[2][2]`，即 `sum += arr[i][i]` |
| scanf 读二维 | `scanf("%d", &matrix[i][j])` — `&` 不能丢 |

## 关键理解

- 二维数组本质上是**数组的数组**：`matrix[i]` 是第 i 行的一维数组 ✅
- 嵌套循环时外层管行、内层管列，顺序不能乱 ✅
- `scanf` 读二维数组元素也需要 `&`（取地址），和普通变量一样 ✅
- `matrix[i][j] = i * j` 是在**运行时**用循环计算赋值，比手动算好写死更通用 ✅
- `int matrix[3][3]` 空声明（不初始化）时，值是不确定的，赋值后才可用 ✅
